SocialGallery Block is a Gutenberg base block plugin that displays Instagram media in website gallery layouts.

== Description ==

SocialGallery Block is a Gutenberg base block plugin that is relying on a 3rd party as a service. This plugin will send requests to the Instagram Graph API to retrieve data and show Instagram media in gallery layouts on websites. This plugin represents the gallery in the grid and masonry layout. It has such control that you can change its style or typo according to your requirements. This plugin has added a gallery to Lightbox which shows the media with a description in Lightbox.

**Service Used:**
- Instagram Graph API: https://developers.facebook.com/docs/instagram-api

**Terms of Use:**
- Instagram Graph API Terms: https://www.facebook.com/legal/terms

**Privacy Policy:**
- Instagram Privacy Policy: https://privacycenter.instagram.com/policy/

By using this plugin, you agree to the Instagram Graph API terms and privacy policy. The plugin will send requests to the Instagram Graph API to retrieve data and may transmit some user data as required by the API.

== Installation ==

SocialGallery Block plugin is very easy to handle and simple. Following few instructions:
1. Install the plugin and activate it.
2. Go to the SocialGallery Block menu and fill the token key into the input text area. Click on the save setting. 
3. After that go to any page/post and add SocialGallery Block.

== Frequently Asked Questions ==

= What is the SocialGallery Block plugin? =

This plugin is based on the Gutenberg wp block editor and is compatible with all WordPress themes. It helps Instagram posts as gallery layout.

= How does this plugin work? =

Instagram access token is required for this plugin.Then it will send requests to the Instagram Graph API to retrieve data and may transmit some user data as required by the API.

= How to get an Instagram Access Token? =

It can get through the Facebook developer site. [Follow this video to get tokens easily](https://youtu.be/uUkl3xZF_pg).
